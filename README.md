# Proyecto-ICM3233-Refrigeración-del-departamento-de-mecánica
En el presente informe se aborda el dimensionamiento del sistema de climatización para las oficinas administrativas, una sala de estudio y un espacio común para profesores del Departamento de Ingeniería Mecánica y Metalúrgica (DIMM) en el contexto de su remodelación.

El documento ANÁLISIS[2].ipynb incluye desde la modelación del "día representativo", hasta el cálculo de los grados-hora y la determinación de la carga térmica de los distintos recintos.

El documento GRÁFICOS FINALES.ipynb incluye la construcción de los gráficos comparativos utilizados en las conclusiones, a partir de los datos calculados de costo para cada una de las alternativas de climatización.
